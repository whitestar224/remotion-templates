# Remotion Lab License Notes

Source: https://remotionlab.com/showcase

Observed public terms as of 2026-06-15:

- Remotion Lab describes itself as a Remotion learning/template resource.
- Its showcase page says users can browse templates and download `.tsx` files.
- Its terms say code templates use MIT licensing and may be used, modified, and distributed, including commercially, with original license notice retained.
- Its terms also say full code/download access requires login and prohibit unauthorized bulk scraping and distribution of member-only content.

## Import rule for this repository

Only commit template files when at least one of these is true:

1. the file is clearly public and redistributable;
2. the downloaded file itself contains an MIT/license header allowing redistribution;
3. you have permission from Remotion Lab to redistribute it;
4. this repository is private and the file is used only for your own learning/personal creation.

When importing any `.tsx` file, keep its original header comments and attribution.

## What this pack does not include

This pack does not include copied Remotion Lab template source code, because the full code was not available from the public page without logging in and should not be bulk-scraped.
