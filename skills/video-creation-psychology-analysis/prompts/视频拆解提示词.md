# 视频拆解提示词

## 提示词 1：完整深层逐镜头拆解

请你作为「视频创作心理分析师 + 口播文案导演 + 动效拆解师」，分析我上传的视频。

要求：

1. 不要只描述画面是什么，要分析为什么创作者偏偏使用这个画面和动效。
2. 每次明显画面、素材、动效、字幕、信息密度或情绪变化，都单独拆成一个视觉单元。
3. 每个单元必须包含：
   - 时间码
   - 口播/文案要点
   - 画面/素材
   - 动效/剪辑
   - 表层作用
   - 深层心理/导演想法
   - 为什么偏偏这样配
   - 为什么不用其他方案
   - 可复用思路
4. 必须对比替代方案，比如：
   - 为什么不用录屏
   - 为什么不用截图墙
   - 为什么不用纯文字
   - 为什么不用真人
   - 为什么不用普通素材图
   - 为什么不用论文/资料堆
5. 最后总结整条视频的视觉语法、素材选择逻辑、动效语言和可复用模板。

输出成一份适合保存的 Markdown 文档。

## 提示词 2：只分析某一个镜头

请你精细分析这个镜头：

时间码：____  
口播句子：____  
画面描述：____  
动效描述：____  

重点回答：

1. 这句话本质上在解决观众什么心理问题？
2. 这个画面为什么能解决这个问题？
3. 这个动效为什么适合这个语气？
4. 如果改成录屏、截图墙、纯文字、真人、素材图，各自会弱在哪里？
5. 这个镜头的可复用公式是什么？

## 提示词 3：生成 Remotion 复刻任务书

请根据这份视频创作心理分析，生成一份 Remotion 复刻任务书。

要求包括：

- 画面尺寸
- 帧率
- 场景分层
- 组件结构
- 每个镜头的入场、停留、退出
- 每个镜头的动效参数
- 字幕与画面对应
- debug 版本要求
- clean 版本要求
- 验收标准

重点：不要只复刻画面，要复刻每个镜头背后的心理功能。
