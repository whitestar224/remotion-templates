# CLAUDE.md — 视频创作心理分析 Skill

## Project Memory

本项目用于视频创作心理分析，尤其是知识类视频、科普视频、口播视频、方法论视频的「口播文案 × 画面 × 动效」深层拆解。

## Claude Code 使用方式

当用户在 Claude Code 中说：

- “拆解这个视频”
- “分析这句文案为什么配这个画面”
- “把画面动效和口播对应起来”
- “生成 Remotion 复刻任务书”
- “做成视频创作分析文档”

请读取本目录下的 `SKILL.md`，然后按其中的字段输出。

## 分析准则

你不能只写：

> 画面是电脑套壳，动效是推入。

你必须写：

> 这句口播本质上在建立创作者经验可信度。电脑套壳让观众进入作者的创作现场；页面截图提供真实证据；推入动效制造开场仪式感。如果换成录屏，会太像教程；如果换成截图墙，会过早变成炫耀成果；如果换成纯文字，则缺少证据感。

## 必须做替代方案比较

每个关键镜头至少对比 3 种替代方案：

- 录屏
- 截图墙
- 纯文字
- 真人讲解
- 普通素材图
- 论文/资料堆

## 输出风格

- 中文输出；
- 结论直接；
- 拆解要细；
- 不要泛泛而谈；
- 多写“为什么偏偏这样做”；
- 最后必须总结成用户能复用的模板。
