# 视频创作心理分析检查清单

## A. 拆解粒度

- [ ] 每次明显画面变化都单独拆了
- [ ] 每次明显动效变化都单独拆了
- [ ] 字幕样式变化被记录了
- [ ] 信息密度变化被记录了
- [ ] 情绪语气变化被记录了
- [ ] 章节转场被单独分析了

## B. 心理分析

- [ ] 写清楚该镜头解决了观众什么心理问题
- [ ] 写清楚创作者为什么这样安排
- [ ] 写清楚观众看完这个镜头应该产生什么感受
- [ ] 写清楚画面与口播之间的隐含关系
- [ ] 写清楚画面是证据、结构、情绪、场景还是升华

## C. 替代方案对比

- [ ] 为什么不用纯录屏
- [ ] 为什么不用截图墙
- [ ] 为什么不用纯文字
- [ ] 为什么不用真人讲解
- [ ] 为什么不用普通素材图
- [ ] 为什么不用论文/资料堆
- [ ] 为什么不用更复杂或更简单的动效

## D. 复用价值

- [ ] 每个镜头都有可复用思路
- [ ] 总结了画面语言
- [ ] 总结了动效语言
- [ ] 总结了素材选择逻辑
- [ ] 总结了口播类型与画面类型的对应关系
- [ ] 给出了 Remotion / AE 可复刻建议

## E. 常见问题

- [ ] 有没有只写“增强可信度”但没说具体怎么增强？
- [ ] 有没有只写“吸引注意力”但没区分注意力类型？
- [ ] 有没有把所有画面都解释成“更生动”？
- [ ] 有没有漏掉“为什么偏偏这样做”？
- [ ] 有没有漏掉“如果换成别的方案会怎样”？
