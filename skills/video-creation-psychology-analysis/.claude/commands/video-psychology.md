# video-psychology

请读取本项目中的：

- `.skills/video-creation-psychology-analysis/SKILL.md`
- `.skills/video-creation-psychology-analysis/templates/逐镜头深层拆解表模板.md`
- `.skills/video-creation-psychology-analysis/checklists/视频创作心理分析检查清单.md`

然后按「视频创作心理分析 Skill」拆解用户提供的视频或片段。

要求：

1. 每次画面/素材/动效/字幕/情绪变化都单独拆成一个视觉单元。
2. 不要只写画面是什么，要写为什么偏偏这样做。
3. 每个关键镜头都要对比替代方案：录屏、截图墙、纯文字、真人、普通素材图、论文堆。
4. 最后输出可复用创作模板和 Remotion / AE 复刻建议。
